BMI Calculator

This project was for my CSCI 430, Mobile Computing course. I worked on it with another classmate. It has a user registration page where you can create an ID and password. Then you are able to sign in using your password. After the disclaimer page, you input your personal info and then you arrive at the main menu selection. You have 4 options to pick from. Update user information, New record, Graph and Suggestions. This application is meant to keep track of your BMI and shows you data and suggestions.
